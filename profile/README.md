## About Kanvas

Kanvas, which is derived from canvas, is named to be where users and developers can draw what they want to draw on.
To achieve this goal as a Ethereum L2, we are trying to pursue following goals.

- Usability
- Simplicity
- EVM-equivalence
- Low fee
- High scalability
- Decentralization
